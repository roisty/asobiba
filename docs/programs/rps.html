仕組み

3つのボタン「グー」、「チョキ」、「パー」が押されたとき、それぞれ「じゃんけん関数」に 0,1,2の引数を渡します。
引数というのは、関数を呼び出す側が「関数名(引数1,引数2…)」のように、関数にデータを渡すために作られているものです。
関数は、以下のコードで引数を取得できます。
function test( a, b…) { //処理 }
この例は、test("GO","GOOD");と呼び出されたときに、変数aにGOが、変数BにGOODが入るような例です。
今回は、変数selectに0,1,2のどれかを渡しています。

次に、勝ち負けリストを用意します。
勝ち負けリストは、配列を使いました。
配列とは、[[0,1,2],[1,2,3],[2,3,4]]のように、数値を簡単に取得できるようにする仕組みです。
配列のは、[0,1,2]のようにも定義でき、["a","b","c"]のようにも定義できます。
今回は、勝ち、あいこ、負けの順で、0,1,2とします。
そうすると、[[1, 0, 2],[2, 1, 0],[0, 2, 1]]のような配列になります。これをjudgmentとします。
配列の中身の取得は「配列名[何個目か]]」の形で取得できるので、judgment[select][機械のランダム]で勝ち、負け、あいこを調べられます。

お疲れさまでした。あと少しです。
最後に、ランダムな数を取得する方法について調べます。
ランダムな数は、以下のコードで取得できます。
Math.random()
でも、これだと0~1のランダムな数が出てくるので、「勝ち」「負け」「あいこ」の数である3をかけて、切り捨てます。
その結果、コードは以下の通りになります。
Math.floor(Math.random() * (3));
これで完成です。
