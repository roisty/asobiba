ストップウォッチ（可能な限り簡単なスクリプトで作成）

仕組み

ボタンが押されたとき、一秒待ち、表示する数字を1上げる関数Stopwatchを作る。
Stopwatchの中で、数字を1上げた後、またStopwatchを呼び出すようにする。
それにより、Stopwatchが1秒ごとにループし、ストップウォッチができる。

ソースコード

HTML

              <!DOCTYPE html>
<html lang="ja">

<head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-1">
    <title>Stopwatch</title>
    <script src="/main.js"></script>
</head>

<body>
    <h1 class="time"></h1>
    <button class="start" onclick="time = 0;stopwatch();">スタート</button>
</body>

</html>>

JavaScript

var time = 0;

function stopwatch() {
  window.setTimeout(function() {
    time++;
    document.querySelector(".time").innerText = time + "秒";
    stopwatch();
  }, 1000)
}
